## 安装Hexo命令到系统
```
$ npm install hexo-cli -g
```

## 初始化
```
$ hexo init
```

## 安装modules
```
$ npm install
```

## 创建文章
```
hexo new name
```

## 创建页面名
```
hexo new page name
```

## 生成页面
```
$ hexo g # hexo generate
```

## 启动本地服务
```
$ hexo s # hexo server
```

## 浏览器查看
```
$ open http://localhost:4000
```
